# Pawfolio

A Pet Health & Vaccination Tracker built with Flask.

## Current Features
- Pet profiles
- Gender tracking
- Weight tracking
- Vaccination status display
- Add Pet page
- GitHub version control

## Planned Features
- Pet registration form
- SQLite database
- Vaccination reminders
- Weight history tracking
- Edit/Delete pets
- Cloud deployment

## Technologies
- Python
- Flask
- HTML
- CSS
- Git
- GitHub
